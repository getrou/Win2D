Win2D documentation
Introduction

Win2D is an easy-to-use Windows Runtime API for immediate mode 2D graphics rendering with GPU acceleration. It is available to C#, C++ and VB developers writing apps for the Windows Universal Platform (UWP) or for with Project Reunion (WinUI3). It utilizes the power of Direct2D, and integrates seamlessly with XAML and CoreWindow.

Win2D currently has two packages available:

  • Win2D.uwp

    • For use with the Windows Universal Platform (WinUI2)

  • Microsoft.Graphics.Win2D

    • For use with ProjectReunion (WinUI3)
